# ClickWise

Discover your **Digital Security DNA** — a behavior-first cybersecurity education platform that measures how you act online, identifies weaknesses, and turns them into personalized learning missions.

## What it does

- **Assessment** — realistic scenarios across 12 security dimensions
- **Digital DNA Profile** — personality-style traits + security score
- **Missions** — progressive challenges from beginner to expert
- **Cyber Games** — phishing, passwords, social engineering, ransomware, and more
- **Emergency Mode** — calm step-by-step guides for real incidents
- **Learning Center** — courses, labs, and daily tips
- **Gamification** — XP, levels, streaks, ranks, and achievements

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- Lucide icons

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Progress is stored in the browser (`localStorage`) for privacy.
